
# Restaurant Management System 

## Overview
This Java application is designed to streamline restaurant operations by providing modules for food menu management, order processing, and customer billing. It includes a modern GUI (built with Swing or JavaFX) and supports database connectivity for robust data storage and retrieval.[3][4][5]

## Features
- Admin and employee login authentication
- Food menu management: add, update, delete dishes, set prices
- Customer order creation and editing
- Real-time order tracking and status update
- Quantity, subtotal, tax, and total bill calculations
- Bill generation and printing
- Sort/filter view for menu items
- Daily sales summary/export
- Simple, intuitive graphical interface

## Technologies Used
- Java SE (Swing/JavaFX)
- MySQL (JDBC) or file-based storage (for smaller projects)
- Eclipse 

## Installation

1. **Prerequisites:**
   - Java JDK installed (recommended version: Java SE 18+)
   - MySQL installed and running
   - Download MySQL Connector/J for JDBC integration

## Usage

- **Login:** Admin/staff access via secure login.
- **Menu Management:** Add or edit food items, prices, descriptions.
- **Order Processing:** Select items from menu, set quantities, assign table/customer, view real-time bill estimates.
- **Billing:** Generate and print invoices for completed orders.
- **Daily Reports:** View total sales, top dishes, and other statistics.

Advanced Features
--------------------
Supports multiple user types: admin, waiter, chef, cashier.
Notification module to alert chef/waiter when an order is placed.
SMS or email integration for sending bills/receipts (requires user mobile/email).
Real-time dashboard for order status, sales analytics, inventory alerts.
